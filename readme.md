# Data Management Mini Toolkit (DMMT)

##  Overview

This project implements three important data structures used in real-world applications:

* Binary Search Tree (BST)
* Graph (BFS & DFS Traversal)
* Hash Table (with Separate Chaining)

The goal is to understand how data is stored, searched, and managed efficiently.

---

##  Features

###  Binary Search Tree (BST)

* Insert elements
* Search elements
* Delete nodes (leaf, one child, two children)
* Inorder traversal (sorted output)

###  Graph

* Representation using adjacency list
* Breadth First Search (BFS)
* Depth First Search (DFS)

###  Hash Table

* Insert key-value pairs
* Retrieve values
* Delete keys
* Collision handling using separate chaining

---

##  Concepts Covered

* Tree traversal and operations
* Graph traversal techniques
* Hashing and collision handling
* Time complexity analysis
